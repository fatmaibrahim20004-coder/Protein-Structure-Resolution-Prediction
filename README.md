 ## Protein-Structure-Resolution-Prediction
## Project Overview
This project is a Machine Learning web application that predicts protein structure resolution based on biochemical features such as Matthews coefficient, solvent content, pH, temperature, R-free, and R-work.
*The model is deployed using Streamlit Cloud.
## Aplication link:
https://protein-structure-resolution-prediction-ufkrnsjxpt99g9yjdmjxlb.streamlit.app/
 ## presentation video:

 ## Features
* Interactive sliders for input parameters
* Real-time prediction using trained ML model
* Easy-to-use web interface
* Visual interpretation of prediction quality
 ## Model Details
* Algorithm: Random Forest Regressor
* Hyperparameter tuning applied
* Features used:
    * Matthews Coefficient
    * Solvent Content
    * pH
    * Temperature
    * R-free
    * R-work
